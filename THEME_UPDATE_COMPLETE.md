## Color Usage Guide

### When to use each color:
- **#F4E1D7 (Cream)**: Page backgrounds, subtle highlights
- **#D96432 (Orange)**: Primary actions, important buttons, active states
- **#70161E (Maroon)**: Headers, navbar, text emphasis
- **#D9A036 (Gold)**: Accents, borders, secondary highlights
- **#4A5D44 (Green)**: Secondary buttons, table headers, borders
